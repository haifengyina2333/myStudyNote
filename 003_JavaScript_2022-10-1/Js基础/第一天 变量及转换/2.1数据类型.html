变量的数据类型

js的变量数据类型是靠程序运行时等号右边来决定的

js是动态语言，数据类型可以变化的

简单数据类型

Number , Boolean , String , Undefined , Null

Number 数字型

八进制 前写 0
var num1 = 010; （0）
十六进制 前面写 0x

var num2 = 0x9; （9）
二进制 前面写 0b

var num3 = 0b10; (2)
最大值
Number.MAX_VALVE
最小值
Number.MIN_VALVE
特殊值
Infinity 无穷大
-Infinity 无穷小
NaN 代表非数值（非数字的，包括字符串）
isNaN（）
如果是数字 返回false ，不是返回true

字符串类型 String

单引号 或者是 双引号
var myname = 'nihao';
如果需要嵌套 就 外单内双 或者是 外双内单

转义字符 （写在单引号里面）

\n 换行
\\ \
\' '
\" "
\t Tab
\b 空格

字符串的长度（参考Java）

使用length属性

字符串拼接（拼在一起一定会变成字符串）

'你好啊'+'！我叫周。'
+ 数值相加 字符相连

字符串拼接加强

可以字符串里 + 变量
'' + 变量 + ''

布尔型 Boolean

true (1) 真
false (0) 假
加法运算当 1,0来看

Undefined

如果一个变量只声明未给值，就是'Undefined'
Undefined + 数字 是 Nan
和字符相连

Null 空值

Null + 数字 是 数字
和字符相连

使用typeof检测数据类型

var num =10 ;
typeof num 会返回number
其他按规则来，除了 null
null会返回 object

prompt方法取过来的数值是字符型的

字面量
